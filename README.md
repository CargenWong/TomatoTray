## 简洁易用的 Windows 番茄钟应用

这是一个基于 Electron 开发的轻量级番茄钟应用，灵感来自 macOS 上的开源软件 TomatoBar。  
支持常驻托盘操作，定时提醒、声音提示、以及自定义番茄时长设置，专为 Windows 用户打造。。

### 主要功能

- 托盘启动和停止番茄钟
- 自定义番茄时长
- 提示音 开 / 关
- 到时弹窗 + 声音提醒

### 安装与运行

1. 克隆项目到本地：
```
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名
```
2. 安装依赖：
```
npm install
```
3. 启动应用：
```
npm start
```
